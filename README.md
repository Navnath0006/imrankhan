# Dr. Imrankhan Jiragal - Personal Portfolio

A professional portfolio website for Dr. Imrankhan Jiragal, showcasing his academic qualifications, research experience, publications, and professional skills.

## Features

- **Modern Design:** Clean, professional interface designed with Tailwind CSS.
- **Dark/Light Mode:** Full support for dark and light themes with a toggle switch.
- **Smooth Scrolling:** Integrated [Lenis](https://github.com/studio-freight/lenis) for a premium smooth scrolling experience.
- **Responsive:** Fully responsive design that works seamlessly on desktop, tablet, and mobile devices.
- **Multi-page Layout:**
    - **Home:** Hero section and introduction.
    - **About:** Detailed summary, education, skills, and awards.
    - **Experience:** Interactive timeline of professional service and research.
    - **Publications:** Comprehensive list of thesis, journals, books, and abstracts.
    - **Contact:** Contact information and form.

## Technologies Used

- **HTML5**
- **CSS3** (Tailwind CSS via CDN)
- **JavaScript** (Vanilla)
- **Lenis** (Smooth Scroll Library)
- **FontAwesome** (Icons)

## Installation & Usage

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Open the project folder.
3.  Open `index.html` in your web browser.

## Customization

- **Styles:** Custom styles and Lenis configuration are located in `css/style.css`.
- **Scripts:** Theme toggle logic and Lenis initialization are in `js/script.js`.
- **Content:** Edit the HTML files (`index.html`, `about.html`, etc.) to update information.
